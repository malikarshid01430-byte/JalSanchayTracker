# 💧 JalSanchayTracker – Android App

A modern Android application built to monitor water conservation activities, rainwater harvesting systems, groundwater levels, and daily water usage. The app focuses on sustainability, data tracking, and a simple, user friendly experience.

---

## 🚀 Features

• 🔐 Secure user registration and login
• 👤 Profile creation and profile update
• 🌐 Multi language support with major Indian languages
• 💧 Daily water consumption tracking
• 🌧️ Rainwater harvesting monitoring
• 📉 Groundwater level records
• 🏞️ Water conservation project management
• 📍 Location based water resource tracking
• 📊 Charts and analytics dashboard
• 🔔 Notifications and water saving reminders
• 📅 Maintenance schedules and alerts
• 📷 Upload photos of conservation projects
• 📝 Reports and historical records
• ❤️ Save favorite locations and projects
• ⚙️ Settings, language switch, and logout
• 📱 Professional dashboard with colorful Material Design UI
• ⚡ Fast and responsive performance

---

## 🛠️ Tech Stack

• Language: Java or Kotlin
• Platform: Android SDK
• Database: SQLite
• UI: XML Layouts with Material Design Components
• Architecture: MVVM
• Charts: MPAndroidChart
• Image Loading: Glide or Picasso
• Notifications: Android Notification Manager

---

## 📂 Project Structure

com.example.jalsanchaytracker

ui/ → Activities, Fragments, and ViewModels
data/ → Models, Repositories, and SQLite helpers
adapter/ → RecyclerView adapters
utils/ → Helper classes and constants
res/ → Layouts, drawables, strings, and themes

2. Click Open.
3. Select the project folder.
4. Wait for Gradle sync to finish.

## 📦 Modules

### LoginActivity

Handles user authentication.

### RegisterActivity

Creates new user accounts.

### MainActivity

Displays the professional dashboard.

### ProfileActivity

Shows and updates user profile details.

### WaterUsageActivity

Tracks daily water consumption.

### RainwaterHarvestingActivity

Monitors rainwater harvesting systems.

### GroundwaterActivity

Stores groundwater level data.

### ConservationProjectsActivity

Manages water conservation projects.

### AnalyticsActivity

Displays charts and statistics.

### NotificationsActivity

Shows alerts and reminders.

### ReportsActivity

Generates detailed reports.

### SettingsActivity

Manages language, theme, and preferences.
